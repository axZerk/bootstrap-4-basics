Что такое Bootstrap?

Фреймворк для разработки с помощью HTML, CSS и JS. Имеет инструментарий с открытым исходным кодом. Позволяет быстро прототипировать свои идеи или создавать полноценный, mobile-first вебсайт с помощью множества готовых компонентов, респонсив сетки и плагинов, построенных на jQuery.

Контейнер

Предоставляет средства для центрирования содержимого страницы.

Используем .container для фиксированной ширины или
.container-fluid для полноценной ширины.

<div class="container"></div>

Строки

Строки .row представляют собой горизонтальные группы столбцов (ячеек), которые обеспечивают правильное выравнивание столбцов (ячеек).

<div class="row"></div>

Колонки/ячейки/столбцы

Контент страницы должен быть помещен в столбцы .col-x-x, и только столбцы могут быть непосредственными дочерними элементами строк.

Классы столбцов указывают количество столбцов, которые вы хотели бы использовать, из возможных 12 в строке. Если нам нужен контент в 4 столбца на маленьких устройствах, мы можем использовать .col-sm-4

<div class="col-sm-4"></div>

Простой layout

Контейнер, в нем строка, в строке 3 ячейки которые занимают по 4 столбца каждая. Таким образом мы получаем 3-х колоночный layout. 

      <div class="container">
      	<div class="row">
      		<div class="col-sm-4">1</div>
      		<div class="col-sm-4">2</div>
      		<div class="col-sm-4">3</div>
      	</div>
      </div>
